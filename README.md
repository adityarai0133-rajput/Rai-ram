# Rai-Ram Optimizer 🚀

Developed by **Aditya Rai** from **Aadi-Tech** (Laksar, Uttarakhand).

## Description
Rai-Ram is a zero-footprint memory management tool designed for legacy x86 hardware and low-end PCs (like i3 with 8GB RAM). It uses **NF-1 (Nano Flash-1)** logic to reclaim system memory.
